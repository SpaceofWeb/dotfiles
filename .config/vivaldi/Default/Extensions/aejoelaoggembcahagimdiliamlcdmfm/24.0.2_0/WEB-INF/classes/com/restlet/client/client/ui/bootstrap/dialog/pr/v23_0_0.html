Deprecating the legacy expression syntax

In previous versions, Restlet Client evaluated expressions with or without a leading $.
For examples: ${host} and {host}.

From now on, API Tester only evaluates expressions with a leading $ (${host}).
Expressions with a leading $ provide a significant performance boost on big JSON payloads.
Note that all your existing expressions have been updated to the dollar syntax.
